# Code of Conduct

This project follows the Contributor Covenant code of conduct.

## Our Pledge

In the interest of fostering an open and welcoming community, we pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the project maintainers.

For more information, see https://www.contributor-covenant.org/version/2/0/code_of_conduct/
